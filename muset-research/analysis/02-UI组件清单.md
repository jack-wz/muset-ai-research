```
├── 顶部导航栏（Top Bar）
├── 左侧边栏（Left Sidebar）
├── 主内容区（Main Content）
└── 右侧聊天面板（Right Chat Panel）
```

## 二、顶部导航栏组件

### 组件列表
1. **Logo区域**
   - Muset Logo图标
   - 可点击返回首页

2. **新建页面按钮**
   - 图标：加号
   - 功能：快速创建新页面

3. **AI额度显示**
   - 显示剩余AI聊天次数

4. **用户头像菜单**

## 三、左侧边栏组件

### 1. 我的页面（My Pages）

```
My Pages

│   ├── "My pages"文本
```

### 2. 社区创作展示（Community Creations）

**创作卡片组件**:
```
创作卡片
├── 类型标签（左上角）
│   ├── 图标
```

## 六、通用UI组件

### 2. 图标系统

## 十四、设置与配置界面组件

### 1. 设置抽屉（Settings Drawer）
```
Settings Drawer
├── 顶部标签页
│   ├── General
│   ├── Models
│   ├── MCP
│   └── Skills
├── 内容区域（随标签切换）
└── 底部操作区
    ├── Test Connection
    └── Save Changes
```
- 通过右上角齿轮按钮呼出
- 支持 ESC 关闭、点击遮罩关闭
- 表单字段使用 Radix Form + controlled inputs

### 2. 模型配置面板
- 字段：模型提供商选择、Model ID、API Key（密文输入框）、Base URL、Streaming 开关
- “Latency preview” 子组件：显示最近一次响应耗时、吞吐
- “Apply to workspace” 切换，决定是否为当前工作区默认模型

### 3. MCP 服务器卡片
- 列表展示所有注册服务器，每项包含：
  - 服务器名称/地址
  - 状态标签（Connected / Error / Disabled）
  - 工具数量
  - 操作按钮：`Connect`、`Disconnect`、`Edit`
- 展开面板显示工具清单，支持逐个启用/禁用

### 4. 技能管理列表
- 两列卡片布局（Skill Gallery）
- 卡片内容：
  - 技能图标 + 名称 + Provider
  - 描述与适用场景标签（Style、Translation、Research等）
  - Toggle 开关（启用/停用）
  - “View manifest” 链接（弹出 SKILL.md 预览）
- 右上角 `Upload skill package` 按钮，触发文件上传模态

## 十五、写作风格与多语言组件

### 1. 风格面板（Style Lab）
```
Style Lab
├── 样本上传区（拖拽上传、支持多文件）
├── 语气滑块（Formal ↔ Creative）
├── 句长/节奏选择器（ButtonGroup）
├── Do / Don't 列表编辑器
└── 实时预览卡片
```
- 上传成功后展示提取到的风格关键词
- 底部 `Apply to current page` 按钮触发后端风格更新

### 2. 语言切换组件
- 位置：顶部导航的用户菜单内
- 形式：带国旗/语言缩写的 `Select`，选项包括 EN、ZH、ES、JA 等
- 切换后触发：
  - UI 字符串即时更新
  - 聊天面板 placeholder 改成选定语言
  - 默认模型/技能随语言提示（例如加载多语言技能包）

## 十六、AI 工具栏与版本对比

### 1. 浮动 AI 工具栏
```
Floating Toolbar
├── 继续写作（Continue）
├── 总结（Summarize）
├── 润色（Polish）
├── 翻译（Translate）
└── 扩展（Expand）
```
- 触发条件：编辑器中选中文本
- 包含模型选择下拉 + “Use style profile” Toggle
- 执行后在右侧显示 diff preview

### 2. 版本历史侧栏
- 时间线布局（Timeline）
- 每个版本节点包含：
  - 版本号 + 时间
  - 触发来源（Auto / Manual / AI Tool）
  - “Compare” 按钮（打开 Diff Modal）
- Diff Modal 结构：
  - 左：旧版本
  - 右：新版本
  - 顶部 tabs：`Raw` / `AI Suggestions`

## 十七、导入导出与安全提示

### 1. 配置导入向导
```
Import Wizard Modal
├── Stepper (Upload → Preview → Apply)
```

- 操作按钮：
  - `Export JSON`
  - `Copy secure link`
- 底部提醒：有效期与脱敏说明
