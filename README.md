# Muset AI 研究项目

这是一个 AI 写作助手项目的研究和设计文档仓库。

## 项目结构

```
muset-research/
├── analysis/           # 产品分析文档
│   ├── 01-产品概述.md
│   ├── 02-UI组件清单.md
│   ├── 03-功能交互流程.md
│   ├── 04-视觉设计规范.md
│   ├── 05-技术实现方案.md
│   └── 06-数据模型设计.md
├── screenshots/        # 界面截图
└── 参考资料.MD

.kiro/specs/ai-writing-assistant/  # 项目规格说明
├── requirements.md     # 需求文档
├── design.md          # 设计文档
└── tasks.md           # 任务清单
```

## 文档说明

- **产品概述**: 项目的整体介绍和目标
- **UI组件清单**: 界面组件的详细列表
- **功能交互流程**: 用户交互流程说明
- **视觉设计规范**: 设计系统和规范
- **技术实现方案**: 技术栈和实现细节
- **数据模型设计**: 数据结构设计

## 开始使用

查看 `muset-research/analysis/` 目录下的文档了解项目详情。
