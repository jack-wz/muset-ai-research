# Muset.ai 视觉设计规范

## 一、设计理念

### 核心原则
1. **简洁优雅**: 去除冗余，突出内容
2. **专业可信**: 体现AI技术的专业性
3. **温暖友好**: 降低使用门槛，亲和力强
4. **创意激发**: 视觉元素激发创作灵感

### 设计关键词
- Clean（简洁）
- Modern（现代）
- Creative（创意）
- Professional（专业）
- Friendly（友好）

## 七、图标系统

- 中等图标: 20px × 20px

### 图标风格

### 图标颜色

## 十一、动画规范

### 过渡时间
```css
--duration-fast: 150ms;
--duration-normal: 200ms;
--duration-slow: 300ms;
```

### 缓动函数
```css
--ease-in: cubic-bezier(0.4, 0, 1, 1);
--ease-out: cubic-bezier(0, 0, 0.2, 1);
--ease-in-out: cubic-bezier(0.4, 0, 0.2, 1);
```

### 常用动画

**淡入淡出**:
```css
transition: opacity var(--duration-normal) var(--ease-in-out);
```

**滑动**:
```css
transition: transform var(--duration-normal) var(--ease-out);
```

**缩放**:
```css
transition: transform var(--duration-fast) var(--ease-out);
```

## 十二、响应式设计

### 断点
```css
--breakpoint-sm: 640px;
--breakpoint-md: 768px;
--breakpoint-lg: 1024px;
--breakpoint-xl: 1280px;
--breakpoint-2xl: 1536px;
```

### 布局适配
- Mobile: 单列，全宽
- Tablet: 两列，侧边栏可折叠
- Desktop: 三列，固定侧边栏

## 十三、可访问性设计

### 对比度
- 正文文字: 至少 4.5:1
- 大文字: 至少 3:1
- 图标: 至少 3:1

### 焦点状态
```css
outline: 2px solid var(--primary-color);
outline-offset: 2px;
```

### 触摸目标
- 最小尺寸: 44px × 44px
- 间距: 至少 8px

## 十四、品牌元素

### Logo使用
- 最小尺寸: 24px
- 安全空间: Logo高度的1/2
- 颜色: 主色或白色

### 吉祥物/插画
- 欢迎表情图
- 引导插画
- 空状态插画
- 错误状态插画

### 图片处理
- 统一圆角: 12px
- 压缩格式: WebP
- 懒加载
- 渐进式加载
