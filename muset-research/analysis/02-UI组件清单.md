# Muset.ai UI组件详细清单

## 一、布局结构

### 1. 主布局（Main Layout）
```
├── 顶部导航栏（Top Bar）
├── 左侧边栏（Left Sidebar）
├── 主内容区（Main Content）
└── 右侧聊天面板（Right Chat Panel）
```

## 二、顶部导航栏组件

### 组件列表
1. **Logo区域**
   - Muset Logo图标
   - 可点击返回首页

2. **新建页面按钮**
   - 图标：加号
   - 功能：快速创建新页面
   - 测试ID：`sidebar-new-page-button`

3. **AI额度显示**
   - 显示剩余AI聊天次数
   - 文本："AI chats left: 35"
   - 升级按钮

4. **用户头像菜单**
   - 圆形头像
   - 下拉菜单
   - 用户设置入口

### 样式特点
- 半透明背景
- 固定定位
- 响应式设计
- 图标统一使用SVG

## 三、左侧边栏组件

### 1. 我的页面（My Pages）
**组件结构**:
```
My Pages
├── 标题栏
│   ├── "My pages"文本
│   └── 更多操作按钮
├── 切换标签
│   └── Chat标签（带图标）
└── 页面列表
    └── Recently Updated轮播
```

**页面卡片组件**:
- 页面图标
- 页面标题
- 页面摘要（最多3行）
- 更新时间
- 操作菜单按钮

**轮播控制**:
- 左右箭头导航
- Slick Carousel实现
- 支持拖拽

### 2. 社区创作展示（Community Creations）

**折叠面板**:
- 标题："Hide Community Creations"
- 展开/收起图标
- 动画过渡

**创作卡片组件**:
```
创作卡片
├── 类型标签（左上角）
│   ├── 图标
│   └── 类型名称
├── 封面图片
│   └── 渐变遮罩
├── 标题
└── 作者信息
    ├── 作者头像
    └── 作者名称
```

**支持的类型标签**:
1. Storyboard Script（故事板脚本）
2. Cartoon Video（卡通视频）
3. Photogen（摄影）
4. Journalism（新闻）
5. Music Video（音乐视频）
6. Rednote（小红书）
7. Commercial Video（商业视频）
8. Short Video（短视频）
9. Comic（漫画）
10. Educational Video（教育视频）
11. Poetry/Translation（诗歌/翻译）

**底部按钮**:
- "Explore Community Creations"
- 带箭头图标
- 点击展开更多

## 四、右侧聊天面板组件

### 1. 顶部工具栏
**按钮组**:
- 关闭按钮
- 最小化按钮
- 更多选项按钮

### 2. 欢迎界面

**欢迎组件**:
```
欢迎界面
├── Logo动画
│   └── Muset欢迎表情图
├── 标语
│   └── "Muset thinks with you"
└── 分隔线
```

### 3. 技能/灵感标签

**轮播组件**:
- 标签："Inspiration"
- 子项目：
  - "See what peers are writing"
  - "Pose a thought-provoking question"
  - "Spark fresh inspiration"

### 4. 输入区域

**输入框组件**:
```
输入容器
├── 左侧工具
│   ├── 添加上下文按钮（@）
│   └── 功能扩展区
├── 中间输入框
│   ├── TipTap编辑器
│   ├── Placeholder: "Drop an idea, let's shape..."
│   └── 支持富文本
└── 右侧工具
    ├── 文件上传按钮
    ├── 图片上传按钮
    └── 发送按钮（禁用状态）
```

**文件上传支持**:
- 文档：txt, pdf, doc, docx, md, csv, xls, xlsx, ppt, pptx, epub等
- 图片：jpg, png, jpeg, gif, bmp, webp

### 5. 提示词建议

```
提示词卡片

```

**示例提示词**:

## 九、响应式设计

### 断点
- Mobile: < 768px
- Tablet: 768px - 1024px
- Desktop: > 1024px

### 适配策略
- 移动端：单列布局
- 平板：两列布局
- 桌面：三列布局
